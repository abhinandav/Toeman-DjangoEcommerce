
# Toeman Shoes Ecommerce⭐💫

Toeman Shoes is a dynamic ecommerce website developed to provide a seamless online shopping experience. Built with HTML and Bootstrap for the front end, and powered by Python with the Django framework for the backend, the website ensures a robust and user-friendly interface.

## Features

- A visually appealing and User-friendly Interface for an enhanced shopping experience

- Secure user authentication with OTP verification during registration and login processes.

- Customized admin panel for efficient management of products, user accounts, and orders

- Secure and smooth payment transactions using Razorpay Integration, ensuring a reliable checkout process

  
## Deployment

Clone the repository to your local machine.

```bash
  git clone https://github.com/abhinz/Toeman-DjangoEcommerce.git
```



Set up your virtual environment.

```bash
  virtualenv env
  source env/bin/activate
```


Install the dependencies.

```bash
  pip install -r requirements.txt
```


Run the Django migrations.

```bash
 python manage.py migrate
```

Start the development server.

```bash
python manage.py runserver
```
